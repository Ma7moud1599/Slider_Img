# Image Slider — JavaScript

A responsive image carousel/slider built with vanilla JavaScript. Supports auto-play, manual navigation, and smooth CSS transitions.

## Features

- Auto-play with configurable interval
- Previous / Next navigation arrows
- Dot indicators with active state
- Smooth CSS slide transitions
- Pause on hover

## Running

Open `index.html` in any browser.

## Tech Stack

Vanilla JavaScript, CSS3 transitions, HTML5
